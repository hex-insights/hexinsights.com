# Hex Insights Website

Repository for the website of Hex Insights.

This site is built with [Gatsby](https://www.gatsbyjs.org/).

## Development

To work on this site, you'll need to install the Gatsby CLI tool, which can be done with `npm install -g gatsby-cli`. To run the development server, navigate to the `client/` directory and run `gatsby develop`, the development server will then be running on `localhost:8000`.

## Deployment

To deploy changes, run the command `npm run deploy` from the `client/` directory. This will create a new commit on the `gh-pages` branch and update the site.
